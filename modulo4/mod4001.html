Formação Basica

escopo da variavel

Variaveis

  • VAR
    • usado para declaração simples de variaveis sem controles maiores
  • LET
    • quando usado o script melhor delimita os escopos, não permitindo duplicidades

Constantes

  • CONST
    • não é possivel alterar os valores definidos em constantes

Objetos

  • Dot Notation
    • curso.titulo curso.titulo:imagem => retorna ERRO
  • Bracket Notation
    • curso["titulo"] curso["titulo:imagem"] => funciona quando se quer transformar o nome da propriedade em string
